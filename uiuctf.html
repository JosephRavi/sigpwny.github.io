ABOUT

UIUCTF is a CTF run by a special interest group named Sigpwny. SIGPwny is a student-run information security club at UIUC. We host weekly public meetings on low-barrier-of-entry security topics that are focused on collaborative learning and friendly environments. In addition, we coordinate weekly focus groups for more in-depth exploration in topics like embedded hacking, penetration testing, and fuzzing research.
